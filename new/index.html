// features

Everything folded into one menu

No bloat, no five different hubs open at once. Blud keeps every tool a session actually needs in a single tabbed window.

01

Universal script hub

Detects the game you're in and loads the right module set automatically — no manual selection.

02

Autofarm & grinding

Set-and-forget farming loops for supported simulator and tycoon-style games.

03

ESP & visuals

Player, item, and chest ESP with adjustable distance and team-color filtering.

04

Movement tools

Speed, noclip, infinite jump, and fly, each with its own keybind and slider.

05

Keybind menu

Rebind anything from the settings tab — no digging through config files.

06

Saved configs

Your toggle states and sliders persist between sessions, per game.
